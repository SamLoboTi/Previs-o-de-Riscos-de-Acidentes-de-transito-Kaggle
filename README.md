Os organizadores da competição reservam-se o direito de atualizar o cronograma do concurso caso julguem necessário. Sobre a Série Tabular Playground 

O objetivo da Tabular Playground Series é oferecer à comunidade Kaggle uma variedade de desafios relativamente leves que podem ser usados para aprender e aprimorar habilidades em diferentes aspectos do aprendizado de máquina e ciência de dados. A duração de cada competição geralmente dura apenas algumas semanas, podendo ter durações maiores ou mais curtas dependendo do desafio. Os desafios geralmente usarão conjuntos de dados relativamente leves, gerados sinteticamente a partir de dados do mundo real, e proporcionarão uma oportunidade de iterar rapidamente por várias ideias de engenharia de modelos e características, criar visualizações, etc. Conjuntos de Dados Gerados Sinteticamente Usar dados sintéticos para competições do Playground nos permite encontrar um equilíbrio entre ter dados do mundo real (com recursos nomeados) e garantir que os rótulos de teste não estejam disponíveis publicamente. Isso nos permite realizar competições com conjuntos de dados mais interessantes do que no passado. Embora ainda existam desafios na geração de dados sintéticos, o estado da arte é muito melhor agora do que quando começamos a Tabular Playground Series há dois anos, e esse objetivo é produzir conjuntos de dados que tenham muito menos artefatos.
